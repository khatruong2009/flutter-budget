# Product Overview

This is a personal budget tracking Flutter application that helps users manage their finances across multiple platforms (iOS, Android, Web, macOS, Linux, Windows).

## Core Features

- **Transaction Management**: Add, edit, and categorize income and expenses
- **Monthly Budgeting**: Track spending and income by month with visual summaries
- **Categories**: Organize transactions by custom categories with pie chart visualization
- **History**: View transaction history across different time periods
- **Data Export**: Export transaction data to CSV format
- **Quick Actions**: iOS/Android shortcuts for rapid expense/income entry
- **AI Chat Integration**: Built-in chat functionality for financial insights
- **Theme Support**: Light/dark mode with system preference detection
- **Data Persistence**: Local storage using SharedPreferences

## Target Platforms

Cross-platform Flutter app supporting iOS, Android, Web, macOS, Linux, and Windows with platform-specific optimizations and native UI patterns where appropriate.
